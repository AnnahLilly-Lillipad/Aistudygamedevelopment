# StudyTales Development Roadmap

Here's what we're working on and what's next for StudyTales! 🚀

## Current Focus (v0.1)

### Core Gameplay
- [x] Character collection system
- [x] Gacha pull mechanics
- [x] Battle system with knowledge questions
- [x] Progression (levels, XP, streaks)
- [ ] **In Progress:** Balance tuning for fair rewards
- [ ] **In Progress:** More character variety

### Features
- [x] Dashboard with stats and quests
- [x] Study mode
- [x] Multiple mini-games
- [x] AI Chat companion
- [x] User profiles
- [ ] **Coming:** Daily challenges
- [ ] **Coming:** Leaderboards
