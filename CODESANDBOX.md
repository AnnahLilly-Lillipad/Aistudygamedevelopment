### What Happens
1. CodeSandbox clones the repository
2. Dependencies are installed automatically
3. The dev server starts on port 5000
4. The preview opens in the browser

## Troubleshooting

### "Command not found: npm"
- CodeSandbox might be using pnpm instead
- It handles this automatically - just wait for the setup

### Port 5000 is already in use
- CodeSandbox will automatically try a different port
- Check the terminal output for the actual URL

### Hot Module Replacement (HMR) issues
- The `vite.config.ts` includes HMR settings for CodeSandbox
- If changes don't reflect, use the refresh button in CodeSandbox

### Out of Memory
- Some features require significant resources
- Try reducing the number of cards rendered or simplifying animations
