# Deploy StudyTales\n\nHere are the easiest ways to deploy StudyTales to the web!\n\n## 🚀 Option 1: Vercel (Recommended - Fastest Setup)\n\n1. **Push to GitHub** (Already done!)\n2. **Go to [vercel.com](https://vercel.com)**\n3. **Click "Add New" → "Project"**\n4. **Select your GitHub repo: `AnnahLilly-Lillipad/Aistudygamedevelopment`**\n5. **Framework: Vite**\n6. **Deploy!**\n\n✅ **Free tier includes:**\n- Custom domain\n- Automatic deployments on push\n- Preview URLs for pull requests\n- HTTPS by default\n\n**Your site will be live at:** `studytales.vercel.app` (or custom domain)\n\n---\n\n## 🌐 Option 2: Netlify (Also Great)\n\n1. **Go to [netlify.com](https://netlify.com)**\n2. **Click "Add new site" → "Import an existing project"**\n3. **Connect GitHub → Select your repo**\n4. **Settings:**\n   - Build command: `npm run build`\n   - Publish directory: `dist`\n5. **Deploy!**\n\n✅ **Features:**\n- Automatic deployments\n- Free SSL certificate\n- Custom domain support\n- Deploy previews\n\n**Your site will be live at:** `studytales.netlify.app`\n\n---\n\n## ☁️ Option 3: GitHub Pages (Free & Simple)\n\nPerfect if you want zero cost and don't need custom domains.\n\n### Already Configured!\nJust enable in your repo settings:\n
1. Go to repo Settings → Pages\n2. Source: Deploy from branch → gh-pages\n3. Click Save\n\n**Your site will be live at:** `https://annahlilly-lillipad.github.io/Aistudygamedevelopment/`\n\n---\n\n## 🐳 Option 4: Docker + Any Host (Advanced)\n\nFor Railway, Render, or Fly.io:\n\n### Create Dockerfile:\n
```dockerfile\nFROM node:18-alpine\n\nWORKDIR /app\n\nCOPY package*.json ./\nRUN npm ci\n\nCOPY . .\nRUN npm run build\n\nENV HOST=0.0.0.0\nENV PORT=5000\n\nCMD ["npm", "run", "preview"]\n```\n\n### Create .dockerignore:\n
```\nnode_modules\n.git\ndist\n.env.local\n```\n\nThen deploy to:\n- **[Railway.app](https://railway.app)** - Push to deploy\n- **[Render.com](https://render.com)** - Free tier available\n- **[Fly.io](https://fly.io)** - Global edge deployment\n\n---\n\n## 📊 Comparison Table\n\n| Platform | Cost | Setup Time | Custom Domain | Auto Deploy |\n|----------|------|-----------|---|---|\n| **Vercel** | Free | 2 min | Yes | Yes |\n| **Netlify** | Free | 2 min | Yes | Yes |\n| **GitHub Pages** | Free | 1 min | Subdomain only | Yes |\n| **Railway** | Free tier | 3 min | Yes | Yes |\n| **Render** | Free tier | 3 min | Yes | Yes |\n\n---\n\n## 🎮 Play Your Game!\n\nOnce deployed, share the URL with friends!
